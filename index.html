Informacion del desarrollo

Newsdata.io es la mejor API de noticias basada en JSON que extrae datos de noticias de más de 10 000 fuentes de noticias confiables en más de 45 idiomas y más de 10 categorías. Newsdata.io viene con una función de búsqueda de noticias, que le permite buscar datos de noticias a través de palabras clave, frases y con filtros de búsqueda avanzada puede filtrar datos de noticias según la fecha y la hora, el país, los idiomas y muchos más para obtener noticias útiles. y puede descargar los datos en formato CSV y XLSX.

Una API de REST, o API de RESTful, es una interfaz de programación de aplicaciones (API o API web) que se ajusta a los límites de la arquitectura REST y permite la interacción con los servicios web de RESTful

Newsdata nos ofrece la Informacion a traves de un archivo JSON, de esta manera podemos extrar facilmente la Informacion y pintarla en pantalla
